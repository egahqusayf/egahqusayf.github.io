Sentiment Analysis of YouTube Comments on the Indonesian Vice President

This project analyzes public sentiment toward the Indonesian Vice President based on 80,000+ YouTube comments using a fine-tuned XLM-RoBERTa transformer model and a custom-built web-based annotation tool.

Python · NLP · Transformers · XLM-RoBERTa · PyTorch · Flask · HuggingFace

Problem & Motivation

Social media platforms such as YouTube play a crucial role in shaping and reflecting public opinion toward political figures. However, the massive volume of user-generated comments makes manual analysis impractical.

This project aims to automatically identify and analyze sentiment patterns in YouTube comments related to the Indonesian Vice President, providing insights into public perception using modern Natural Language Processing techniques.

Dataset Overview

  • Source: YouTube Data API v3
  • Total Comments: 80,000+
  • Language: Indonesian (with mixed slang & regional terms)
  • Labels: Negative, Neutral, Positive

Manual Annotation Process

Due to the absence of reliable pre-labeled datasets for Indonesian political discourse, a custom web-based annotation tool was developed using Flask to manually label comments.

  • Pagination-based labeling (20 comments per page)
  • Real-time sentiment statistics
  • Temporary in-memory labeling before saving
  • Export to Excel/CSV format

Modeling Approach

The sentiment classification task was addressed using XLM-RoBERTa Base, a multilingual transformer model capable of handling Indonesian language and informal text patterns.

  • Model: xlm-roberta-base
  • Task: Multiclass classification (3 classes)
  • Framework: HuggingFace Transformers & PyTorch
  • Training: Fine-tuning on manually annotated data

Large-Scale Inference

After fine-tuning, the model was used to perform inference on tens of thousands of unlabeled comments. Batch inference was executed on GPU to improve computational efficiency.

Each prediction includes both a sentiment label and a confidence score derived from softmax probabilities.

Visualization & Analysis

The distribution of predicted sentiments reveals the overall tone of public discourse toward the Vice President. Confidence scores were also analyzed to assess prediction reliability.

Key Insights

  • Public sentiment on YouTube shows a mix of strong opinions and neutral discussion.
  • Transformer-based models handle informal political discourse effectively.
  • Manual annotation significantly improves model reliability compared to zero-shot methods.

Limitations & Future Work

Some challenges include sarcasm, regional languages (e.g., Javanese), and code-mixed text. Future work may involve domain-adaptive pretraining, larger annotated datasets, or comparison with IndoBERT variants.
