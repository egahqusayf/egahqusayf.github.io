# Design References

This revision adapts interaction and presentation principles from two high-scoring 2025 portfolio examples reviewed on CSS Design Awards.

## 1. Kaito Note — Portfolio

- Published CSSDA aggregate score: **8.17**
- Reference: https://www.cssdesignawards.com/sites/kaito-note/47392/
- Relevant qualities: animated grid, scroll-led interaction, strong visual identity, and restrained high-impact color.
- Adapted in this portfolio through: pointer-reactive glow, tilt depth, animated focus text, image-led process cards, and a more expressive project grid.

## 2. Michele Du — Portfolio

- Published CSSDA aggregate score: **7.98**
- Reference: https://www.cssdesignawards.com/sites/michele-du/48375/
- Relevant qualities: dashboard-style portfolio, data-driven UX, interactive systems, and visual storytelling.
- Adapted in this portfolio through: project quick-view panels, visible metrics, evidence-first case studies, and dashboard-oriented project imagery.

The implementation does not copy either website. It translates the relevant principles into a lightweight static portfolio that remains suitable for GitHub Pages.
